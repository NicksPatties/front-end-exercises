Thoughts

The original solution to this design uses divs to create the digits, which is not a semantically correct solution. The smarter way to handle this would be to create an SVG, or use an image for a logo, and include an aira-label instead.

However, to match the styles, I used dev tools to find the dimensions of the elements, which seems like cheating... I should find a tool that allows me to find pixel distances. This information would probably be visible in tools like Figma or Photoshop, so I don't think it's that too bad.

Improvements

I made some enhancements to this little project compared to the original.

  • Added semantically correct HTML elements. I didn't forget the <span>100</span>, so screen readers and text based sites can see it.
  • I added a dark mode color scheme.
